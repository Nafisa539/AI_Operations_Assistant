# AI_Operations_Assistant
AI-Powered Operations Assistant Project
 
## Project Description
This project is designed to assist operations teams in managing online academic programs. The system provides a conversational interface to answer student queries, retrieve schedule information, generate reminder emails, and analyze student activity data.
 
## Objectives
- Answer frequently asked student questions.
- Provide academic schedule information.
- Generate reminder and notification emails.
- Analyze student attendance and assignment status.
- Improve operational efficiency through automation.
 
## Features
- FAQ-based chatbot
- Schedule lookup
- Student activity analytics
- Email generation
- Conversational interface
 
## Technologies Used
- Python
- HTML
- CSS
- JavaScript
- Flask / FastAPI
- CSV Datasets
- GitHub
